Chapter 1.1 - Sets: Complete Questions & Detailed Solutions

Q1: Which of the following are sets? Justify your answer.

(i) Collection of months starting with 'J': Yes, it is a set: {January, June, July}.

(ii) Ten most talented writers: Not a set; "talented" is subjective.

(iii) Eleven best cricket batsmen: Not a set; "best" is subjective.

(iv) All boys in your class: Yes, it is a well-defined set.

(v) Natural numbers less than 100: Yes, {1, 2, ..., 99}.

(vi) Novels by Munshi Prem Chand: Yes, the collection is well-defined.

(vii) Even integers: Yes, {..., -4, -2, 0, 2, 4, ...}.

(viii) Questions in this chapter: Yes, the collection is definite.

(ix) Most dangerous animals: Not a set; "dangerous" is subjective.

Q2: Specify the following sets using the roster form.

(i) A = {x : x is an integer, -3 ≤ x < 7} ⇒ {-3, -2, -1, 0, 1, 2, 3, 4, 5, 6}

(ii) B = {x : x is a natural number less than 6} ⇒ {1, 2, 3, 4, 5}

(iii) C = {x : x is a two-digit number with digit sum 8} ⇒ {17, 26, 35, 44, 53, 62, 71, 80, 89, 98}

(iv) D = {x : x is a prime divisor of 60} ⇒ {2, 3, 5}

Q3: Write the following sets in set-builder form.

(i) E = {0, 3, 6, 9, 12} ⇒ E = {x : x = 3n, n ∈ {0, 1, 2, 3, 4}}

(ii) F = {3, 6, 9, 12, 15} ⇒ F = {x : x = 3n, n ∈ {1, 2, 3, 4, 5}}

(iii) G = {1, 4, 9, 16, 25, 36} ⇒ G = {x : x = n², n ∈ {1, 2, 3, 4, 5, 6}}

(iv) H = {2, 4, 8, 16, 32} ⇒ H = {x : x = 2ⁿ, n ∈ {1, 2, 3, 4, 5}}

Q4: State whether the following sets are finite or infinite.

(i) Set of months in a year: Finite

(ii) Set {1, 2, 3, ..., 100}: Finite

(iii) Set of positive integers greater than 100: Infinite

(iv) Set of prime numbers less than 99: Finite

Q5: List all the elements of the following sets.

(i) A = {x : x is a natural number and x² ≤16} ⇒ {1, 2, 3, 4}

(ii) B = {x : x is an integer, -2 < x ≤3} ⇒ {-1, 0, 1, 2, 3}

(iii) C = Letters in "TRIANGLE" ⇒ {T, R, I, A, N, G, L, E}

(iv) D = Divisors of 18 ⇒ {1, 2, 3, 6, 9, 18}

Q6: Match each of the following with the correct set form.

(i) {a, e, i, o, u} ⇒ Set of vowels in the English alphabet.

(ii) {2, 4, 6, 8, 10} ⇒ Even natural numbers ≤10.

(iii) {Sunday, Monday, Tuesday, Wednesday, Thursday, Friday, Saturday} ⇒ Days of the week.

(iv) {Jan, Feb, Mar, Apr, May, Jun, Jul, Aug, Sep, Oct, Nov, Dec} ⇒ Months of the year.
